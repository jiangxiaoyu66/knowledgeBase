# 盒模型



- `content-box` (W3C 标准盒模型)：盒子大小就是content大小
- `border-box` (IE 盒模型)：盒子大小还包括padding和border

可通过`box-sizing`进行设置。

不写box-sizing的话，写了DOCTYPE类型的，使用content-box；不屑就使用border-box
